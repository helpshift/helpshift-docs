</Intro>

## 言語の設定 {#setting-a-language}

<SetLanguageDescription />

例:

<Tabs>

<TabItem value="Objective-C Example" label="Objective-Cの例">

|                |                                                                                                                                                                                             |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 定義           | <code>setLanguage:</code>                                                                                                                                                                   |
| サンプルコード | <code> [Helpshift setLanguage:"fr"];</code>の<code>@"fr"</code>は言語コードです。次のような形式で言語コードや国コードを渡すこともできます。<code> [Helpshift setLanguage:@"zh-SG"]; </code> |

</TabItem>
<TabItem value="Swift Example" label="Swiftの例">

|                |                                                                                                                                                                                        |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 定義           | <code>setLanguage()</code>                                                                                                                                                             |
| サンプルコード | <code> Helpshift.setLanguage("fr")</code>の<code>"fr"</code>は言語コードです。次のような形式で言語コードや国コードを渡すこともできます。<code> Helpshift.setLanguage("zh-SG"); </code> |

</TabItem>

</Tabs>

言語をフランス語に設定した後の SDK の画面のスクリーンショットです。

![](/static/books/sdkx_ios/ios-Internationalization.png)

<Admonition type="info" title="注意">

デフォルトの言語は英語（`en`）に設定されています。すなわち、`setLanguage:` API を使用して言語を設定していない場合には、SDK は英語を使用します。

- 言語コードは、すべて大文字と小文字を区別します。

</Admonition>
