# ライセンス情報 {#licensing}

<Intro>

サードパーティのライセンス情報です。

</Intro>

## CWStatusBarNotification

[CWStatusBarNotification](https://github.com/cezarywojcik/CWStatusBarNotification)はアプリ内通知用の UI を表示するために使用されます。

## Reachability

[Reachability](https://github.com/tonymillion/Reachability)は、デバイスのネットワーク状態を確認するために使用されます。

## QBImagePicker

[QBImagePicker](https://github.com/nek023/QBImagePicker)は、i（Pad）OS 13 以前のバージョンを実行中のデバイスでイメージピッカーを表示するために使用されます。
