# ライセンス情報 {#licensing}

<Intro>

サードパーティのライセンス情報です。

</Intro>

## CWStatusBarNotification {#CWStatusBarNotification}

[CWStatusBarNotification](https://github.com/cezarywojcik/CWStatusBarNotification)はアプリ内通知用のUIを表示するために使用されます。

## Reachability {#Reachability}

[Reachability](https://github.com/tonymillion/Reachability)は、デバイスのネットワーク状態を確認するために使用されます。

## QBImagePicker

[QBImagePicker](https://github.com/nek023/QBImagePicker)は、i（Pad）OS 13以前のバージョンを実行中のデバイスでイメージピッカーを表示するために使用されます。

注意: QBImagePickerはSDK Xバージョン10.3.xでのみ使用可能です。
