# インパクト分析 {#performance}

<Intro>

SDKのリリースごとに、このページではSDKがアプリケーションのビルドと実行時間のさまざまな側面に与える影響を追跡します。

</Intro>

<Admonition type="info" title="注意">

**アプリの起動時間**

Helpshiftの初期化はすべてバックグラウンドのスレッドで実行されるため、Helpshift SDKがアプリの起動時間に影響を与えることはありません。

</Admonition>

# バージョン別のレポート {#version-impact}

Helpshift SDKの各バージョンのインパクト分析レポートについては、以下でご確認いただけます。

## 10.4.0

<Admonition type="info" title="注意">

macOS 14.6.1、Xcode 16.0.0、iOS 18.1.1（iPhone 15）の環境で実施しています。メモリの使用量は、会話内のメッセージ数に応じて増加します。

</Admonition>

### ディスク容量

xcframeworkが占有するディスク容量です。

xcframeworkのサイズは、デバイスが7.2MB、シミュレーターが8.1MBです。合計サイズは15.3MBです。

内訳は以下の通りです。

| アーキテクチャ | デバイスのサイズ（MB） | シミュレーターのサイズ（MB） |
| ------------ | ---------------- | ------------------- |
| arm64        | 0.85             | 0.83                |
| x86_64       | NA               | 0.77                |

iOSアプリが`HelpshiftX.xcframework`と統合された場合、IPAファイルサイズの増加は約`647`KBです。

### RAMの使用量

|                                                                   |                                                                                   |
| ----------------------------------------------------------------- | --------------------------------------------------------------------------------- |
| **Helpshiftのチャット画面（<code>showConversation:</code>）**      | <code>4.1</code>MBが追加されます（チャット画面上のメッセージの数に依存します）。     |
| **HelpshiftのFAQ画面（<code>showFAQs:</code>）**              | <code>3.6</code>MBが追加されます。                                                         |
| **HelpshiftのFAQセクション画面（<code>showFAQSection:</code>）** | <code>3.5</code>MBが追加されます。                                                         |
| **Helpshiftの単一のFAQ画面（<code>showFAQ:</code>）**         | <code>3.3</code>MBが追加されます。                                                         |
