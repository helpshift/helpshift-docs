# デザイン

<Intro>

Helpshiftの使用体験をカスタマイズします。

</Intro>

## ブランディングとカスタマイズ {#branding-customization}

「ダッシュボードの設定」の**「ブランディングとカスタマイズ」ページ**では、あなたのアプリのスタイルやテーマに合わせてSDKのテーマやスタイルを簡単にカスタマイズできます。

<Image src="/static/books/helpshiftx/branding_customization.png" width="full" />{" "}

アプリのブランディングとカスタマイズに関する詳細については、[**こちら**](https://support.helpshift.com/kb/article/configure-look-and-feel-settings/)をご確認ください。
