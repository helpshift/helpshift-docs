# インパクト分析 {#performance}

<Intro>

以下の番号は、Helpshift SDKと統合された枠組みとなるサンプルアプリ
を示しています。

</Intro>

## v10.4.0

### アプリ内のSDKペイロード {#payload-size}

Helpshift SDK Xは、アプリケーションのリリースAPKに`246 Kb`を追加します。
この分析は、Android Gradle Plugin `v8.1.0`とGradle `v8.1`を使用して実行されました。
`246 Kb`というサイズには、すべてのリソースとコードを含む影響の合計値が含まれています。

### メソッドの数 {#methods-count}

Helpshift SDK Xは、あなたのdexに1765件のメソッドを追加します。

| パッケージ           | メソッドの数 |
| ----------------- | -----------: |
| **com.helpshift** |       `1765` |

### メモリの使用量 {#memory-usage}

| Helpshift APIの起動                   | Javaヒープメモリ（単位: MB） | 総メモリ使用量（単位: MB） |
| -------------------------------------- | ------------------------------- | -------------------------------: |
| showFAQs                               | `6.35`                          |                         `102-110` |
| showFAQSection                         | `5.15`                          |                          `92-102` |
| showSingleFAQ                          | `4.75`                          |                          `85-95` |
| showConversation（会話モード） | `7.55`                          |                            `132-138` |

このメモリへの影響は、アプリが実行されているデバイスのOSおよびメモリ構成にも依存します。

<Admonition type="info" title="注意">

Android 15を搭載したGoogle Pixel 6上のデモアプリでテストされています。

</Admonition>

| パッケージ           | メソッドの数 |
| ----------------- | -----------: |

### メモリの使用量 {#memory-usage}

`Helpshift.showConversation (Myactivity.this, configMap)` APIを呼び出してHelpshiftの会話画面を開くと、
約20 - 25MBのメモリが消費されます。
このメモリへの影響は、アプリが実行されているデバイスのOSおよびメモリ構成にも依存します。

<Admonition type="info" title="注意">

Android 10が搭載されたSamsung Galaxy M11上のデモアプリでテストされています。

</Admonition>
