# Getting Started {#getting-started}

## Introduction {#intro}

<Image src="/static/books/web-chat/wc-embedded.png"  width="major" centered />

## Before you begin {#prerequisites}

In your Helpshift Dashboard, make sure you've added `Web Chat` as a platorm supported by your App. [Here's how to do it.](https://support.helpshift.com/kb/article/web-chat#section-one)


## **Step 1:** Add the Web Chat embed code  {#add-embed-code}

You can find a prepared embed code for your App at `Settings` > `App` > `Web Chat`. <Image src="/static/books/web-chat/wc-embed-code.png"  width="full" /> 
Place the embed code just before the closing `</body>` tag of your page.

[Skip to next step](#start-using) if you've done this.

### Manually adding the embed code {#manual-embed}

In case you don't have a prepared embed code, add this snippet just before the closing `</body>` tag of your page:

```
<script>
  (function () {
    var PLATFORM_ID = "<YOUR_PLATFORM_ID>",
        DOMAIN = "<YOUR_DOMAIN>";

    window.helpshiftConfig = {
      platformId: PLATFORM_ID,
      domain: DOMAIN
    };
  }) ();

  !function(t,e){if("function"!=typeof window.Helpshift){var n=function(){n.q.push(arguments)};n.q=[],window.Helpshift=n;var i,a=t.getElementsByTagName("script")[0];if(t.getElementById(e))return;i=t.createElement("script"),i.async=!0,i.id=e,i.src="https://webchat.helpshift.com/webChat.js";var o=function(){window.Helpshift("init")};window.attachEvent?i.attachEvent("onload",o):i.addEventListener("load",o,!1),a.parentNode.insertBefore(i,a)}else window.Helpshift("update")}(document,"hs-chat");
</script>

```

1. Replace `YOUR_DOMAIN` with your Helpshift domain.
For example, _happyapps_ if your dashboard sits at _happyapps.helpshift.com_
1. Replace `YOUR_PLATFORM_ID` with details found at `Settings` > `SDKs (for Developers)`

## **Step 2:** Start using Helpshift Web Chat  {#start-using}

Helpshift Web Chat is now available on your website. Test it out by starting a chat.

## Supported Browser Versions {#browser-support}
The Helpshift Web Chat widget currently supports the latest versions of following browsers:

* Chrome
* Firefox
* Safari
* Edge
* Mobile Safari
* Mobile Chrome
