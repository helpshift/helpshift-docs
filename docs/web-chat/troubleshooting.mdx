# Troubleshooting

## Responsive Mobile View

The following meta tag should be added in the head of the client website, and the website should be mobile friendly in order to have the perfect chat experience on the mobile device:

```
<meta name="viewport" content="width=device-width, initial-scale=1">
```

## Local Storage

[Local storage](https://developer.mozilla.org/en/docs/Web/API/Window/localStorage) is a prerequisite for the Web Chat widget. If your user has disabled local storage, the Web Chat widget will not work.
