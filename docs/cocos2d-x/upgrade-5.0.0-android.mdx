# Upgrading to 5.0.0 and above

<Intro>

In order to use SDK 5.x for your app, Helpshift recommends you to follow the below steps. For any queries or feedback, please <a href="https://support.helpshift.com/kb/contact/" target="_blank">Contact Us</a>

</Intro>

<Admonition type="danger" title="Important">

If you are looking to upgrade from a version older than 5.x, Make sure to go through the upgrade guides for 1.6.x, 1.7.x and 1.9.x

</Admonition>

- Dowload the latest SDK from [here](/cocos2d-x/getting-started-android#getting-sdk)

- Remove all integration related to Campaigns feature.

- Fix all compilation errors related to Campaings integration code.
