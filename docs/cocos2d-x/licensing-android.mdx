# License Information {#licensing}

## Android Support Library {#AndroidSupportLibrary}

Following Android Support libraries are used to build SDK UI :

- Appcompat library (android-Support-v4 and android-Support-v7).
- Cardview library.
- Recyclerview library.
- Design library.

## Android Volley Library {#AndroidVolleyLibrary}

From [Android Volley](https://github.com/mcxiaoke/android-volley/tree/master/src/main/java/com/android/volley) library, following files are modified and used :

- PoolingByteArrayOutputStream.java
- ByteArrayPool.java
- HttpStatus.java
- HurlStack.java
- Response.java
- BasicNetwork.java
- ExecutorDelivery.java
- ResponseDelivery.java
- JsonObjectResponseParser.java
- JsonArrayResponseParser.java
- NetworkResponse.java
- HttpHeaderParser.java
- Network.java

## Apache Commons Codec {#ApacheCommonsCodec}

From [Apache Commons Codec](https://commons.apache.org/proper/commons-codec/apidocs/overview-summary.html) source code, following files are modified and used :

- [DoubleMetaphone.java](https://commons.apache.org/proper/commons-codec/apidocs/src-html/org/apache/commons/codec/language/DoubleMetaphone.html)
- Encoder.java
- EncoderException.java
- StringEncoder.java

## CircleImageView {#CircleImageView}

From [CircleImageView](https://github.com/hdodenhof/CircleImageView) library, following file is modified and used :

- [CircleImageView.java](https://github.com/hdodenhof/CircleImageView/blob/master/circleimageview/src/main/java/de/hdodenhof/circleimageview/CircleImageView.java)

## AdjustableImageView {#AdjustableImageView}

From [AdjustableImageView](https://github.com/nuuneoi/AdjustableImageView) library, following file is modified and used :

From [NV-websocket-client](https://github.com/TakahikoKawasaki/nv-websocket-client) library v2.2, all classes have been modified and used.
