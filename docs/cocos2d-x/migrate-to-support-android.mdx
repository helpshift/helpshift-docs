# Migration Guide to Support SDK


<Admonition type="info" title="Note">

If you are upgrading to Support SDK then we highly recommend you to upgrade to our [latest version](/cocos2d-x/getting-started-android/)


</Admonition>



Please follow the below steps to migrate to Support SDK:

* Remove the existing integration of the Helpshift Cocos2dx SDK
* Remove all your integration code related to Campaigns feature.
* Import the Helpshift Support SDK package.
* Fix compilation errors if any.
* Test your app for Helpshift integration.
