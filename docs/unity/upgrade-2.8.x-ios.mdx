# Upgrading from 2.8.0 to 2.9.0 {#upgrade-to-2}

<Intro>

The Helpshift plugin v2.9.0 is a major update. Please note that we have deprecated some APIs. If you have questions or feedback, please <a href="https://support.helpshift.com/kb/contact/" target="_blank">Contact Us</a>

</Intro>

## List of deprecated APIs {#deprecated-api}

| Deprecated API                                  | New API                                           |
| ----------------------------------------------- | ------------------------------------------------- |
| `HelpshiftCampaigns.GetCountOfUnreadMessages()` | `HelpshiftCampaigns.RequestUnreadMessagesCount()` |
