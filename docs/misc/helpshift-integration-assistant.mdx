# Helpshift Integration Assistant {#integration-assistant}

## Select the SDK version {#select-sdk}

Select the SDK version from the version dropdown or just give it the path of an SDK that you have.

![integrator-select-sdk.png](/static/books/ios/integrator-select-sdk.png)

## Select your Xcode project {#select-project}

Select the path of your Xcode project.

![integrator-select-project.png](/static/books/ios/integrator-select-project.png)

## Select Localization languages {#select-project}

Select all the languages that you want to support.

![integrator-select-languages.png](/static/books/ios/integrator-select-languages.png)

<Admonition type="danger" title="Important">

### Add only required languages from HSLocalization

If you are adding strings for a certain language, it enables that language in your app.
This has app wide implications and may cause unwanted problems if not taken care of.
Make sure to only add those languages that your app supports.

</Admonition>

## Add theming support {#select-project}

If you want to add theming support to the SDK, just click on `'Yes'`

![integrator-select-theming.png](/static/books/ios/integrator-select-theming.png)

## Integrate {#select-project}

Click on `Add to Project` and relax.

![integrator-success.png](/static/books/ios/integrator-success.png)

## Troubleshooting {#troubleshooting}

If you have trouble running the Integration Assistant, you might need to update your Security and Privacy settings. Follow [steps here](/ios/troubleshooting#integration-assistant-settings) to fix this issue.
