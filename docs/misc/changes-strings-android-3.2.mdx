# hs\_\_strings.xml changes from SDK v3.1.1 to SDK v3.2.0

## Added strings {#added-strings}

|                                             |                                                                                                                                                          |

| <code>hs\_\_sending_fail_msg</code>         | Status text below customer's chat bubble when their reply is not delivered                                                                               |
| <code>hs\_\_mark_yes_no_question</code>     | Text that shows below every single faq to mark the faq as helpful or unhelpful. <Image src="/static/books/android/i18n-mark-yes-no.png"  width="full" /> |
| <code>hs\_\_mark_yes</code>                 | Text for mark helpful button in single faq screen.                                                                                                       |
