# Getting Started

## Requirements {/* #version-requirements */}

- Unity 2022 and above recommended.

### Get the latest Windows PC Widget - Unity SDK {/* #getting-sdk */}

Download the latest Windows PC Widget Unity SDK from the link below:

<DownloadButton
  text="Download SDK"
  link="[% pc-widget-unity.sdk_url %]"
  classes="mb-4"
/>

<Intro>
  Follow these simple steps to add Helpshift in-game support to Unity game right away:
</Intro>

#### Importing the package into the unity project

1. In a game, we can import the package by navigating to **Assets→ Import Package → Custom Package**.
<Image src="/static/books/pc-widget-unity/integrate-1.png" width="half" />

2. Import all the files from the package into the project.
<Image src="/static/books/pc-widget-unity/integrate-2.png" width="half" />

This standard approach allows game developers to control the Helpshift PC Widget's lifecycle using the provided set of [Helpshift PC Widget APIs](/pc-widget-unity/helpshift-apis/). 

## Start using Helpshift {/* #start-using */}

Helpshift is now integrated in your app and ready to collect meaningful data about your users.

### Initializing Helpshift in your app {/* #initializing */}

1. First, create an app on the Helpshift Dashboard
<Image src="/static/books/helpshiftx/add_app.png" width="full" />

2. Create an app with **Windows** as a selected `Platform`
<Image src="/static/books/pc-widget-unity/select_platform.png" width="full" />
