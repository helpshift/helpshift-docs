# Overview {#overview}

<Intro>

Follow this guide if you want to customize emails sent to your customers
to match your branding.

</Intro>

## Email Support Channels {#support-channel}

All the emails sent from Helpshift to your customers can be customized
via Email Support Channels. To view, edit or create your email support
channels, navigate to `Settings`>`Email Support`.

<Image
  src="/static/books/email-support/settings-email-support.png"
  width="full"
/>

<div className=" compact">

|              |                                                              |
| ------------ | ------------------------------------------------------------ |
| <kbd>1</kbd> | Goto Settings                                                |
| <kbd>2</kbd> | Click <code>Email Support</code> from the side navigation    |
| <kbd>3</kbd> | This is where you can edit or add new email support channels |

</div>

## Customizing your Emails {#customizing}

An email support channel allows you to customize the following -

<Image src="/static/books/email-support/email.png" width="full" />

<div className=" compact">

|              |                                                     |
| ------------ | --------------------------------------------------- |
| <kbd>1</kbd> | <code>From name</code> used in the emails           |
| <kbd>2</kbd> | <code>From address</code> used in the emails        |
| <kbd>3</kbd> | The design of the email itself via custom templates |

</div>

In addition, an email support channel let's you match a web/email issue
to itself and customize the auto-reponse and the email template to be used.

<Image
  src="/static/books/email-support/support-channel.png"
  width="full"
/>

<div className=" compact">

|                                   |                                                                                                                                           |
| --------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| <kbd>1</kbd> Auto-response        | Setup an auto-response text for web/email issues matching this channel                                                                    |
| <kbd>2</kbd> Forwarding addresses | Emails forwarded to `support@<your-domain>.mail.helpshift.com` will be associated with this support channel                               |
| <kbd>3</kbd> App association      | If a customer files an issue from your web support portal, and selects this app, the resulting issue is matched with this support channel |
| <kbd>4</kbd> Custom template      | Turn this ON if you wish to use a custom html/text email template for emails sent to your customers.                                      |

</div>
