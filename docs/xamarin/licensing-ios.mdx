<a id="CWStatusBarNotification"></a>
## CWStatusBarNotification

[CWStatusBarNotification](https://github.com/cezarywojcik/CWStatusBarNotification) is used to display the UI for in-app notifications.

<a id="MBProgressHUD"></a>
## MBProgressHUD

[MBProgressHUD](https://github.com/jdg/MBProgressHUD) is used to display a translucent progress indicator.

<a id="JOBadgeView"></a>
## JOBadgeView

[JOBadgeView](https://github.com/radelcom/JOBadgeView) is used to show a the red exclamation indicator when a user's message fails to deliver.

<a id="Reachability"></a>
## Reachability

[Reachability](https://github.com/tonymillion/Reachability) is used to check the device network status.

<a id="SocketRocket"></a>
## SocketRocket

[SocketRocket](https://github.com/facebook/SocketRocket) is used for web sockets.

<a id="PocketSocket"></a>
## PocketSocket

[PocketSocket](https://github.com/zwopple/PocketSocket) is used for per message deflate in web sockets.
