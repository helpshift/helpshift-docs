# Migrating from 1.x to 2.x <small>Android</small> {#migration-guide}

<Intro>

If you have been using the Helpshift Xamarin plugin version 1.x for android and want to upgrade to the 2.x version, please follow the steps below &mdash;

</Intro>

1. Remove all the traces of Helpshift Xamarin SDK from your app. Remove the Helpshift dll reference from you application.

1. See the [Getting Started section](/xamarin/getting-started-android#getting-sdk) to integrate the latest version.
