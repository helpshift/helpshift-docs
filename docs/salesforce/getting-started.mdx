# Getting Started

Helpshift for Salesforce is Helpshift’s new managed package for Salesforce. It gives Salesforce users the ability to add a mobile support channel to their Salesforce Service Cloud by providing native access to a variety of Helpshift features such as live messaging, Knowledge Base mapping, and more.

To get started with the Helpshift plugin for Salesforce, please review our [Salesforce Integration Guide](https://support.helpshift.com/kb/article/salesforce-integration-guide/).
